Firmware on things like NICs, HBAs, disk drives, graphics cards, etc., can create

compatibility issues when other things are updated if not in sync

Vulnerabilities are constantly being discovered, and one of the best defenses

is to keep systems patched and up-to-date

Applications, OS and firmware/drivers should be kept in sync as much as

possible (refer to hardware compatibility matrix) for most systems
